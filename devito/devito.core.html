Submodules¶

devito.core.autotuning module¶

devito.core.autotuning.autotune(operator, arguments, parameters, tunable)[source]¶

Acting as a high-order function, take as input an operator and a list of operator arguments to perform empirical autotuning. Some of the operator arguments are marked as tunable.

devito.core.operator module¶

class devito.core.operator.Operator[source]¶

Bases: object

Module contents¶

The core Devito backend is simply a “shadow” of the base backend, common to all other backends. The core backend (and therefore the base backend as well) are used to run Devito on standard CPU architectures.

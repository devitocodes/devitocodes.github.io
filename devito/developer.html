Developer documentation¶

This page provides an introduction into the internal workings of Devito and is aimed at new developers and people who want to contribute to the project. We will give a brief glimpse under the hood to provide some context for the various design and implementation choices.

If you are only interested in learning how to use Devito, please start with the tutorials and examples.

Please note that this page is currently still under ongoing development.
